# Breadboard Power Supply — 3.3V Regulator (USB-C Powered)

A simple open-hardware power supply module designed to provide a clean and stable **3.3 V** output for breadboard prototyping.  
Powered directly from a **USB-C 5 V** input.

The project is made in **KiCad 9** and includes an **interactive HTML BOM** for quick assembly and part identification.

---

## Features
- USB-C 5 V input (VBUS)
- 3.3 V LDO regulator output
- Compact PCB size, optimized for breadboard use
- Easy-to-source components
- Interactive BOM (`ibom.html`) included
- Open-hardware design (licensed under CERN OHL v1.2)

---

## Electrical Specifications

| Parameter | Value |
|----------|--------|
| Input voltage | 5 V (USB-C VBUS) |
| Output voltage | 3.3 V |
| Regulator type | LDO (linear) |
| Typical output current | depends on LDO (typically 300–800 mA) |
